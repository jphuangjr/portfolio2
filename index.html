TextBER

Uber Rides for ALL.

Textber was a hackathon project that allowed for a new way to request UBER rides without a smartphone. The user would just need to associate their uber account with our service at www.textber.co . After, all you need to hail a UBER car is a phone with text capability!

TEXTBER allows the user to get a fare estimate, and hail a UBER driver once the user provides a starting point and a end address. Built using a Node.js server, UBER's API, Twilio API, and Google Maps API.

Cajamon

Waiting for you flight is now a game.

Cajamon was a Hackathon project created at SFO for the THack Hackathon sponsered by Tnooz, IBM, Amadaeus, Sabre, and SFO Airport. Cajamon is a geo-fenced game inspired by Pokemons. As users encounter new villains, they must battle through to receive special offers from nearby vendors. The potential here is such that families could stay occupied during extended waits or terminal businesses could have a new way to incentivize interactions.

Pi Alpha Phi Rush

Rush site for Pi Alpha Phi of Univeristy of California, Irvine.

PAPHIRUSH.com is a Rush Website currently used by the Pi Alpha Phi Fraternity at the University of California, Irvine as a rush tool to the members keep log and organize the rushees that attend the events. Member are able to rush around campus using either a computer, touch screen pad, or phone to log the information of potential new member as well as keep track of what event each rushee attends. The site was made in the LAMP stack using PHP, Codeigniter, and MySql.

  • Link: www.paphirush.com
  • Client: Pi Alpha Phi, Eta Chapter
  • Technology: LAMP Stack
